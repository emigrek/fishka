# 🐟 Fishka
Flashcard app for learning new words.

## 📦 Used packages
| 📦 Package  | 📋 Reasons |
| ------------- | ------------- |
| Nuxt | Vue framework  |
| Vuetify  | Material design  |
| Axios | Fetching |
| Vuex | State management |
| nanoid | Generating unique ids |

## 🚀 Running
```
git clone https://github.com/emigrek/fishka
cd fishka
npm install
```

Run development server
```
npm run dev
```
or
run production build
```
npm run build
```
